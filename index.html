Návrh a realizácia in-browser aplikácie pre editáciu markdown dokumentov s využitím technológie Web Workers


  1. Stručne charakterizujte markdown jazyk a technológiu Web Workers.
  2. Analyzujte procesy pre editáciu dokumentov a špecifikujte katalóg používateľských požiadaviek.
  3. Vytvorte funkčný a dátový model aplikácie s využitím vybraných diagramov UML.
  4. Navrhnite používateľské rozhranie aplikácie a systém implementujte.
  5. Vypracujte dokumentáciu.
